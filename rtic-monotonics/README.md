# `rtic-monotonics`

> Core abstractions of the Real-Time Interrupt-driven Concurrency (RTIC) Monotonics timers

`rtic-monotonics` is for RTIC v2.

For RTIC v1 see [`rtic-monotonic`](https://github.com/rtic-rs/rtic-monotonic)
