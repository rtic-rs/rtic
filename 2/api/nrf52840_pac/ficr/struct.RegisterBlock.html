0x10 - Code memory page size

0x14 - Code memory size

§deviceid: [DEVICEID; 2]

0x60..0x68 - Description collection[n]: Device identifier

§er: [ER; 4]

0x80..0x90 - Description collection[n]: Encryption root, word n

§ir: [IR; 4]

0x90..0xa0 - Description collection[n]: Identity Root, word n

§deviceaddrtype: DEVICEADDRTYPE

0xa0 - Device address type

§deviceaddr: [DEVICEADDR; 2]

0xa4..0xac - Description collection[n]: Device address n

§info: INFO

0x100..0x120 - Device info

§prodtest: [PRODTEST; 3]

0x350..0x35c - Description collection[n]: Production test signature n

§temp: TEMP

0x404..0x448 - Registers storing factory TEMP module linearization coefficients

§nfc: NFC

0x450..0x460 - Unspecified

§trng90b: TRNG90B

0xc00..0xc20 - NIST800-90B RNG calibration data
