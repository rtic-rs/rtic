pub struct MULTI_CHAN_TRIGGER_SPEC;
Expand description

Trigger one or more channels simultaneously

You can read this register and get multi_chan_trigger::R. You can reset, write, write_with_zero this register using multi_chan_trigger::W. You can also modify this register. See API.
